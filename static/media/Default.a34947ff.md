# markdown-previewer

## Preview text converted to markdown

Place in-line `<Code />` between two back-ticks

```
// For multi-line code, use three consecutive back-tics to open and to close:

let a = 1;
let b = 2;
let c = a + b; // c = 3
```


Want to style text?
- \** : **bold**
- \_ : _italics_
- \*\*_ : **_bold&italics_**
- \~~ : ~~strikethrough~~
- \> : 
> Block quotes
